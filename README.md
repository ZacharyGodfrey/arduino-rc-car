# arduino-rc-car

A simple remote-controlled car. The car receives input through an RF transceiver and uses an Adafruit Motor Shield to control 4 DC motors connected to the wheels.

## Hardware

- 1 x [Arduino Uno Rev3](https://store.arduino.cc/usa/arduino-uno-rev3)
- 1 x [Adafruit Motor Shield for Arduino v2 Kit](https://www.adafruit.com/product/1438)
- 1 x [NRF24L01+PA+LNA Wireless RF Transceiver Module](https://www.amazon.com/HiLetgo%C2%AE-NRF24L01-Wireless-Transceiver-Compatible/dp/B00WG9HO6Q)
- 2 x [3-6V DC Motors with Wheels (2 pack)](https://www.amazon.com/Gearbox-Motor-Wheel-Arduino-Smart/dp/B07P6QCJPX)
- 4 x [M3x35mm Screws (2 pack)](https://www.homedepot.com/p/M3-0-5-x-35-mm-Internal-Hex-Metric-Socket-Head-Cap-Screw-2-Pack-844518/204283594)
- 2 x [M3 Lock Washers (4 pack)](https://www.homedepot.com/p/Everbilt-M3-4-Piece-Stainless-Steel-Metric-Lock-Washer-842398/204993665)
- 4 x [M3 Hex Nuts (2 pack)](https://www.homedepot.com/p/Everbilt-M3-5-Stainless-Steel-Metric-Hex-Nut-2-Piece-per-Bag-842318/204836105)
- 1 x [Inline Power Switch with 2.1mm Barrel Jack](https://www.adafruit.com/product/1125)
- 1 x [6 AA Battery Box with 2.1mm Barrel Jack](https://www.amazon.com/Battery-Holder-Connector-Output-Pack/dp/B07XP7H9PB)
